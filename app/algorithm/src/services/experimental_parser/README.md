## Это экспериментальная версия парсера. На данный момент нестабильна.
Парсер разбит на отдельные файлы.

В файле `test.py` находится простой тест для проверки корректности работы.

В файле `reference.json` находится "эталонный" граф.

В качестве точки доступа используется функция `analyze_code` из файла `parser.py`

Для сборки нескольких графов воедино используется `GraphMerger` из файла `graph_merger.py`

Пример запуска парсера можно посмотреть в тестах.
